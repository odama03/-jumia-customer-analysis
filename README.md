# Jumia Nigeria — Customer Analysis & Segmentation

## Overview
This project analyzes customer behavior data from Jumia Nigeria to 
identify top customers, detect customers at risk of churning and 
segment customers by their purchasing patterns using RFM Analysis.

## Tools Used
- Python
- Pandas
- Plotly
- Dash

## Dataset
250 rows of customer records including total orders, total spent, 
days since last order, loyalty score, refund rate, device used 
and customer status.

## Dashboard Features
- KPI cards showing Total Customers, Total Revenue, Avg Loyalty and Avg Refund Rate
- Customer Status breakdown donut chart (Active, At Risk, Churned)
- Customers by Segment bar chart
- Total Spent by State horizontal bar chart
- Device Used donut chart
- Loyalty Score vs Total Spent scatter chart
- Top 10 Customers by Total Spent bar chart
